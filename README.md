# Read-On

Read On este o aplicatie care isi propune sa imbunatateasca viteza de citire si sa sporeasca confortul cititorului, aducand o serie de imbunatatiri fata de cititul clasic.

### Asseturi folosite:
* TextMesh Pro
* Color Picker
* iTextSharp
* Unity Standalone File Browser
